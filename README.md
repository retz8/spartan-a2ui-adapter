# spartan-a2ui-adapter

An [A2UI](https://github.com/google/a2ui) catalog adapter for [Spartan UI](https://spartan.ng) â€” maps Spartan's Angular component library to the A2UI protocol so agents can reason about and compose Angular UIs.

## What's in this repo

- `libs/spartan-a2ui-adapter/` â€” the adapter library (`@spartan-a2ui-adapter`)
- `libs/spartan-a2ui-adapter/catalogs/v0.8.0/catalog.json` â€” agent-side JSON Schema catalog (published with the npm package)
- `agents/mock/` â€” Google ADK Python agent that generates A2UI responses from natural language prompts
- `apps/mock/` â€” Angular client that sends prompts to the agent via A2A and renders the live response

## Quick start

### 1. Build the adapter library

```bash
npm install
npx nx build spartan-a2ui-adapter
```

### 2. Start the agent

```bash
cd agents/mock
cp .env.example .env
# Edit .env and set GEMINI_API_KEY=your_key_here

uv sync
uv run python -m spartan_mock_agent
```

### 3. Start the Angular app

In a separate terminal:

```bash
npx nx serve mock
```

Open `http://localhost:4200`, type a prompt like `show me a button and badge`, and see Spartan UI components rendered live from the agent.

## Library usage

### 1. Register the catalog

```typescript
import { provideA2UI } from '@a2ui/angular';
import { SPARTAN_CATALOG, SPARTAN_CATALOG_ID } from '@spartan-a2ui-adapter';

export const appConfig: ApplicationConfig = {
  providers: [
    provideA2UI({ catalog: SPARTAN_CATALOG, theme: yourTheme }),
  ],
};
```

### 2. Add a Tailwind source path

Spartan helm components (generated by `npx nx g @spartan-ng/cli:ui`) contain the Tailwind utility classes used by this adapter. Tailwind's automatic content detection does not scan library directories by default, so you need to add an explicit `@source` in your global stylesheet pointing to wherever those components live:

```css
/* styles.css */
@import "tailwindcss";
@source "path/to/your/libs/ui/**/*.ts";  /* adjust to your Spartan UI directory */
@import "@spartan-ng/brain/hlm-tailwind-preset.css";
```

Without this, the adapter renders the correct DOM structure but components appear unstyled.

### 3. Pass the catalog ID to your agent

Pass `SPARTAN_CATALOG_ID` to your agent so it loads the right catalog:

```
https://github.com/retz8/spartan-a2ui-adapter/blob/main/libs/spartan-a2ui-adapter/catalogs/v0.8.0/catalog.json
```
